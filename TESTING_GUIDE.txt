SECTION 4: ROLE HIERARCHY
================================================================================

ROLE_SUPER_ADMIN
  - Full access to everything
  - Can promote users to admin
  - Can remove admins and users
  - Sees all applications, reports, decisions

ROLE_ADMIN
  - Can view all applications
  - Can request documents from applicants
  - Can verify/reject documents
  - Can approve/reject loan applications
  - Can remove regular users only

ROLE_USER
  - Can register and login
  - Can create/submit loan applications (4-step wizard)
  - Can upload documents
  - Can track application status (timeline)
  - Cannot access admin pages

SECTION 7: EMI CALCULATOR (LANDING PAGE)
================================================================================

Formula: EMI = P × r × (1+r)^n / ((1+r)^n - 1)
  P = Principal (loan amount)
  r = Monthly interest rate (annual rate / 12 / 100)
  n = Tenure in months

Use sliders or type values:
  Loan Amount   : ₹10K to ₹50L
  Interest Rate : 1% to 30%
  Tenure        : 6 to 360 months

Shows: Monthly EMI, Principal Amount, Total Interest, Total Payment

================================================================================

Password:
  - Minimum 8 characters

SECTION 11: COMMON ERRORS AND FIXES
================================================================================

ERROR: Registration failed / 403 on register
FIX  : Expired JWT token in browser
       Run in browser console: localStorage.clear()
       Then refresh and try again

ERROR: 401 Unauthorized
FIX  : Token expired or not logged in
       App will auto-logout and redirect to login page

ERROR: 403 Forbidden
FIX  : Wrong role for the endpoint
       Make sure you are logged in as admin for admin features

ERROR: 404 Not Found
FIX  : Service not running or wrong URL
       Check: docker ps
       Check: http://localhost:8761 (all services should be UP)

ERROR: 400 Bad Request
FIX  : Validation failed
       Check password meets requirements
       Check phone is 10 digits starting with 6-9
       Check loan purpose is exact value (HOME not Home)

ERROR: Failed to submit decision
FIX  : Decision already exists for this application
       Each application can only have one decision
       Use a new application for testing

ERROR: ECONNREFUSED / Cannot connect to server
FIX  : Backend not running
       Run: .\start.ps1
       Wait 2-3 minutes for all services to start

ERROR: Services not showing in Eureka
FIX  : Services still starting up
       Wait 1-2 more minutes
       Check: docker logs <service-name> for errors

SECTION 12: API ENDPOINTS REFERENCE
================================================================================

All endpoints go through: http://localhost:8080/gateway/

PUBLIC (no auth needed):
  POST /gateway/auth/register              - Register user
  POST /gateway/auth/register/admin        - Register admin (needs X-Admin-Secret header)
  POST /gateway/auth/register/super-admin  - Register super admin (needs X-Admin-Secret header)
  POST /gateway/auth/login                 - Login
  GET  /gateway/auth/validate              - Validate token
  GET  /gateway/products                   - Get loan products

USER (JWT required):
  POST /gateway/applications               - Create draft application
  PUT  /gateway/applications/{id}          - Update draft
  POST /gateway/applications/{id}/submit   - Submit application
  GET  /gateway/applications/my            - My applications
  GET  /gateway/applications/{id}/status   - Application status
  POST /gateway/documents/upload           - Upload document
  GET  /gateway/documents/my              - My documents

ADMIN (JWT + ROLE_ADMIN or ROLE_SUPER_ADMIN):
  GET  /gateway/admin/applications         - All applications
  GET  /gateway/admin/applications/{id}    - Application detail
  POST /gateway/admin/applications/{id}/decision - Make decision
  GET  /gateway/admin/reports              - Reports/stats
  GET  /gateway/admin/users                - All users
  PUT  /gateway/documents/admin/{id}/verify - Verify document
  GET  /gateway/documents/admin/{id}/view  - View document file
  DELETE /gateway/auth/admin/users/{id}    - Remove user
